#### ✅ 2단계 기능 구현 목록

- [x] 메인화면에 저장한 영상들을 표시
- [x] 저장된 영상이 없으면, 빈 상태를 표시
- [x] 영상 카드에 이모지 버튼(✅, 🗑️)을 추가
  - ✅ 클릭시 본 영상으로 변경
  - 🗑️ 클릭시 저장된 리스트에서 제거 (삭제 확인창 추가)
- [x] 본 영상, 볼 영상을 누르면 필터링
- [x] 디바이스 가로길이에 따른 반응형 적용
  - 1280px 이하: 4개
  - 960px이상~1280px 미만: 3개
  - 600px이상~960px 미만: 2개
  - 600px 미만: 1개
- [x] E2E 플로우 테스트 추가
